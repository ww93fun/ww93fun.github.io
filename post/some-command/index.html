一些命令

Jan 5, 2023

Linux

ssh

ssh会话克隆，不用重复输入密码

编辑 ~/.ssh/config

Host *
    ControlMaster auto
    ControlPath ~/.ssh/%h-%p-%r
    ControlPersist yes

参考自 让mac的终端实现克隆会话功能

文件传输

从服务器上下载文件

scp root@192.168.2.1:/path/filename /local_path

上传本地文件到服务器

scp /local_path/filename root@192.168.2.1:/path

从服务器下载整个目录

scp -r root@192.168.2.1:/remote_path（远程目录） /local_path

上传目录到服务器

scp  -r /local_path root@192.168.2.1:/remote_path

网络

获取服务器外网IP

curl cip.cc
curl ip.cip.cc
curl ifconfig.io

Git

只下载最新代码

git clone --depth=1 https://github.com/xxx/xxx.git
